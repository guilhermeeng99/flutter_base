# FlutterBase 🐦

> Learn Flutter the fun way — interactive quizzes, daily challenges, and gamified learning, Duolingo-style.

---

## ✨ About

**FlutterBase** is an interactive Flutter learning platform. The goal is to make learning Flutter progressive and engaging, with gamification mechanics like XP, daily streaks, levels, and daily challenges — all inside a Flutter app.

### Features

- 🎯 **Interactive quizzes** — multiple choice questions with instant feedback
- 📈 **XP system** — earn points for every correct answer
- 🔥 **Daily streak** — keep your study streak alive day after day
- 🏆 **Level progression** — Beginner → Intermediate → Advanced → Expert
- ⚡ **Daily challenge** — a fresh challenge every day to test your knowledge
- 📊 **Progress dashboard** — track accuracy, total XP, and completed levels
